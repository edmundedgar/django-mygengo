About the myGengo Django Plugin

This Django plugin integrates with the myGengo API to offer low-cost, high-quality translation.

You can download and contribute to the LGPL-licensed myGengo Django Plugin project.

Current Features:

  • Order a Job
    • add text to translate and set source and target languages
    • configure job options and set tier
    • real-time price quote

  • Job Dashboard
    • view worker translations
    • view and post comments
    • post ranslation ratings

Roadmap:

  • accounts and API key settings
  • job grouping
  • more job feedback features
  • e-mail notifications
  • refactoring and polishing the rough edges
