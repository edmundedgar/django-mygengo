About the myGengo Django Plugin

This Django plugin integrates with the myGengo API to offer low-cost, high-quality translation.

You can download and contribute to the LGPL-licensed django-mygengo project.

Note: Until API key information can be edited from the application, set the appropriate values to DEFAULT_MYGENGO_CONFIG in api.py.

Current Features:

  • Order a Job
    • add text to translate and set source and target languages
    • configure job options and set tier
    • real-time price quote

  • Job Dashboard
    • view translations submitted by workers
    • view and post comments
    • post translation ratings

Roadmap:

  • accounts with saved preferences and API key settings
  • job grouping
  • more job feedback features
  • e-mail notifications
  • example blog integration (publish posts in different languages)
  • refactor, improve plugin usability, polish the rough edges
